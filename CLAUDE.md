# Trend Report System

## 基本ルール
- すべての出力は日本語
- reports/daily/YYYY-MM-DD/ に保存
- 作業完了後は必ずgit commit & push
- シンプルさを最優先。複雑にしない

## リポジトリ構成
- topics.json：監視トピック一覧
- agents/：各エージェントへの指示
- reports/daily/：日報
- reports/weekly/：週次まとめ
- feedback/：フィードバック履歴

## エージェント構成
リーダーが以下を並列で指揮する：
- リサーチャー（テック・理系視点）
- リサーチャー（文系・バイタリティ視点）
- リサーチャー（否定的・リスク視点）
- テスター（品質チェック）
- 開発者（アプリ改善）

## 情報収集のルール
- 各トピックについて必ずWeb検索を行う
- 検索は「トピック名 最新情報 今月」で実施
- 検索結果から信頼できる記事を3件以上参照する
- 情報源のURLも記録する
